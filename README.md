# Mappa Interattiva della Rete Bicipolitana del Comune di Pesaro

Questo progetto ospita una mappa GIS interattiva della **Bicipolitana di Pesaro**, pensata per consultare facilmente la rete ciclabile urbana e i servizi associati. L'applicazione è stata sviluppata partendo da dati vettoriali elaborati in **QGIS** e successivamente ottimizzati per il web tramite il plugin `qgis2web`.

## 🚲 Cosa contiene la mappa
* **Tracciati Ciclabili:** Tutte le linee ufficiali (1-12, A, B) navigabili e interrogabili.
* **Servizi e POI:** Posizione esatta di parcheggi bici, noleggi, ciclofficine, fontanelle pubbliche e parchi.
* **Strumenti Interattivi:** * Legenda a comparsa per accendere/spegnere i singoli livelli.
  * Finestre a comparsa (Popup) con i dettagli di ogni elemento cliccato.
  * Strumento righello per misurare le distanze direttamente sulla mappa.
  * Geolocalizzazione GPS per vedere la propria posizione rispetto alle piste.
